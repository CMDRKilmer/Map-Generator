# 六边形地图生成器 — UI 设计预览

采用**暗色游戏主题**，营造沉浸式策略游戏地图编辑器的氛围。

## 组件样式

### 按钮

**普通按钮**
```
┌─────────────────┐
│  普通操作        │  背景: #3a3a4a, 边框: #4a4a5c
└─────────────────┘
```

**主要按钮（生成地图）**
```
┌─────────────────┐
│  🎲 生成地图     │  背景: #5080c8, 边框: #6090d8
└─────────────────┘      悬停: #6098e0
```

**危险按钮**
```
┌─────────────────┐
│  删除           │  背景: #a04040, 边框: #b05050
└─────────────────┘
```

### 滑块

```
海平面: ━━━━●━━━━━━━━━━━━ 35%
        ↑蓝色进度条   ↑圆形滑块（悬停放大）
```

### 复选框

```
☑ 网格线    ☑ 河流
☑ 道路      ☑ 聚落
```

选中状态：蓝色背景 + 白色对勾
未选中状态：深色背景 + 灰色边框

### 下拉框

```
┌─────────────────┐
│ 生物群落    ▼   │  背景: #2a2a38, 边框: #4a4a5c
└─────────────────┘
```

### 分组框

```
┌─ 🗺️ 地图生成 ─────────────┐
│                           │
│   内容区域                 │  标题: #78b4ff (蓝色)
│                           │  边框: #373748
└───────────────────────────┘
```

## 地图渲染区域

### 背景
- 颜色：`#16161c`（深蓝灰色）
- 营造深邃的海洋/夜空感

### 六边形样式

**水域**
- 深海：`#195abf`（深蓝色）
- 浅水：`#3282d4`（亮蓝色）
- 海岸：`#eed696`（沙滩色）

**陆地**
- 平原：`#8cc85a`（绿色）
- 森林：`#3ca03c`（深绿色）
- 沙漠：`#f0d278`（黄色）
- 山地：`#a08c6e`（棕色）
- 雪地：`#f0f0ff`（白色）

### 特性覆盖层

**河流**：蓝色线条 `#468ce6`
**道路**：棕色线条 `#a07850`
**聚落**：
- 村庄：橙色圆点 `#dc8c3c`
- 城镇：红色圆点 `#c83c3c`
- 城市：深红圆点 `#b42828`
- 首都：金色圆点 `#dcb43c` + 星标

## 交互反馈

### 悬停效果
- 按钮：背景变亮，边框高亮
- 滑块：圆形滑块放大，颜色变亮
- 复选框：边框变亮
- 六边形：半透明白色覆盖层

### 选中效果
- 六边形：黄色半透明边框 `#ffff0078`

### 编辑模式
- 光标变为十字准星
- 可编辑六边形显示虚线边框

---

## 状态栏信息

```
📍 坐标: (12, -5)  🌍 ocean ⛰️ 0.32 🌊 水域 │ ⬡ 1081 │ 🌊 水域: 452 │ 🌍 陆地: 629
```

- 坐标：当前鼠标悬停的六边形坐标
- 地形：生物群落类型
- 高程：海拔高度（0-1）
- 特性：水域/河流/聚落/资源等标记
- 统计：总六边形数、水域数、陆地数

---

## 响应式设计

### 最小尺寸
- 窗口：1200 x 800
- 地图区域：600 x 400
- 参数面板：280 x 400

### 自适应规则
- 窗口缩小时，地图区域优先保持比例
- 参数面板固定宽度，可折叠
- 状态栏信息根据宽度自动隐藏次要信息

---

## 动画效果

### 地图生成
- 生成按钮点击后显示旋转加载图标
- 地图数据加载完成后淡入显示

### 六边形交互
- 悬停：100ms 淡入高亮效果
- 选中：200ms 缩放脉冲效果

### 面板切换
- 编辑模式切换：300ms 滑动动画
- 图层切换：200ms 淡入淡出
